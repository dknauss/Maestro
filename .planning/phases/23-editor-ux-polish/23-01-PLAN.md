<objective>
Remove the semantic-colour border system (UX-12 verdict = remove) and convert the
bottom toolbar to native wp-admin idioms (UX-13, toolbar surface): quiet
menu-native icon buttons, a neutral mode chip, Gutenberg-style muted save status,
a non-colour modified marker, and a destructive-text-link Reset All. Colour is
reserved for exactly two meanings — errors and destructive actions, both red.

Purpose: this is the mechanical core of the native-wp-admin restyle. It deletes
the grey/green/amber/red outlined-box vocabulary and replaces it with core's
quiet-control and muted-status idioms, on which every later plan builds.

Output: restyled bottom toolbar (controls + mode chip + save status + modified
marker + Reset All), muted-status i18n strings, and node:test coverage confirming
the status-state mapping seam is intact.
</objective>
